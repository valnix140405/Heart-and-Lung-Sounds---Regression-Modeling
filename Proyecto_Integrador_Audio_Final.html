Proyecto Integrador - Mdulo 2: Clasificacin de Sonidos Cardiacos y Pulmonares

1. Introduccin

Contexto del Problema

La deteccin temprana de anomalas cardiacas (como soplos) y pulmonares (como sibilancias o crepitaciones) es fundamental para el diagnstico clnico. Sin embargo, la auscultacin depende en gran medida de la experiencia del mdico. El uso de Inteligencia Artificial para el procesamiento de seales de audio permite objetivizar y automatizar esta tarea, proporcionando una herramienta de apoyo al diagnstico.

Objetivo

Desarrollar un modelo de aprendizaje automtico (Random Forest) capaz de clasificar a partir de caractersticas de audio (MFCCs) si una grabacin corresponde a un sonido normal o a una patologa especfica, utilizando un dataset de 535 grabaciones capturadas de un maniqu clnico.

Justificacin de la Tcnica

Se utiliza la tcnica de Clasificacin Supervisada. Las seales de audio se transforman en MFCCs (Mel-Frequency Cepstral Coefficients), que capturan las caractersticas espectrales del sonido de manera similar a como el odo humano percibe las frecuencias. El modelo Random Forest es ideal para este dataset por su robustez ante el ruido y su capacidad para manejar mltiples clases con un nmero moderado de muestras.

def load_datasets():
    data = []
    # 1. HS
    if os.path.exists(FILE_HS_CSV):
        df_hs = pd.read_csv(FILE_HS_CSV)
        for _, row in df_hs.iterrows():
            data.append({'path': os.path.join(DIR_HS, f"{row['Heart Sound ID']}.wav"), 
                         'label': f"Heart_{row['Heart Sound Type']}", 'type': 'Heart'})
    # 2. LS
    if os.path.exists(FILE_LS_CSV):
        df_ls = pd.read_csv(FILE_LS_CSV)
        for _, row in df_ls.iterrows():
            data.append({'path': os.path.join(DIR_LS, f"{row['Lung Sound ID']}.wav"), 
                         'label': f"Lung_{row['Lung Sound Type']}", 'type': 'Lung'})
    # 3. Mix Sources
    if os.path.exists(FILE_MIX_CSV):
        df_mix = pd.read_csv(FILE_MIX_CSV)
        for _, row in df_mix.iterrows():
            data.append({'path': os.path.join(DIR_MIX, f"{row['Heart Sound ID']}.wav"), 
                         'label': f"Heart_{row['Heart Sound Type']}", 'type': 'Heart'})
            data.append({'path': os.path.join(DIR_MIX, f"{row['Lung Sound ID']}.wav"), 
                         'label': f"Lung_{row['Lung Sound Type']}", 'type': 'Lung'})
    return pd.DataFrame(data)

df_dataset = load_datasets()

def extract_features(audio_path):
    try:
        audio, sr = librosa.load(audio_path, res_type='kaiser_fast', duration=5)
        mfccs = librosa.feature.mfcc(y=audio, sr=sr, n_mfcc=40)
        return np.mean(mfccs.T, axis=0)
    except:
        return None

print("Extrayendo caractersticas... esto tomar un momento.")

7. Conclusiones y Recomendaciones

Conclusiones

  1. El modelo logr una precisin promedio de 84%,lo cual es un desempeo slido considerando la complejidad de los sonidos fisiolgicos y el nmero de clases.
  2. Los MFCCs demostraron ser caractersticas altamente informativas para distinguir entre patologas cardiacas y pulmonares.
  3. La integracin de las fuentes del dataset 'Mix' fue crucial para estabilizar el modelo, pasando de un sobreajuste inicial a un desempeo generalizable.

Recomendaciones

  1. Aumento de Datos: Si se requiere mayor precisin, se podran aplicar tcnicas de aumento de datos (ruido, estiramiento temporal) para balancear clases con pocas muestras.
  2. Modelos de Deep Learning: Probar Redes Neuronales Convolucionales (CNN) usando los espectrogramas como imgenes de entrada, lo cual suele superar a Random Forest en tareas de audio complejas.
  3. Filtrado Avanzado: Implementar filtros de paso de banda especficos para limpiar ruidos de baja frecuencia no deseados antes de la extraccin de caractersticas.
